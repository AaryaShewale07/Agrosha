# Agrosha
# 🌾 Agrosha – Smart Crop Yield Prediction Platform

**Agrosha** is an intelligent web application designed to assist farmers, agri-entrepreneurs, and researchers by predicting **crop yield** using key agricultural inputs. Our platform not only forecasts yield but also provides estimated **investment** and **profit** insights — making agri-decisions smarter and data-driven.

---

## 🚀 Features

- 🌱 **Yield Prediction** based on:
  - Crop
  - Year
  - Season
  - State
  - Area

- 🌦️ **Auto-fetches critical agri-data** like:
  - Annual Rainfall
  - Fertilizer Usage
  - Pesticide Usage

- 💰 **Estimates Investment and Profit**
  - Know how much you'll spend and earn before you sow!

- 🌐 **Multilingual Support**
  - Built-in **Language Translator** to make Agrosha accessible to users across India.

- 🤖 **Smart Chatbot**
  - Get real-time assistance and information through an integrated chatbot.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Flask, Express.js  
- **Machine Learning Models**: Scikit-learn  
- **Database**: MongoDB  
- **Other Tools**: Git LFS, Multer (for file uploads), GitHub
